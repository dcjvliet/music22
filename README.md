# music22
An open-source project based off of the [music21 repository](https://github.com/cuthbertLab/music21) created by Michael Scott Asato Cuthbert. Since he no longer seems to maintain the project, I wanted to try to create a new and improved version of it myself.

### FFmpeg is required to run music22
Installation instructions can be found [here](https://www.ffmpeg.org/download.html).
